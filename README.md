This project can be built with Maven

Maven command:
```
$ mvn clean install
```

pom.xml entry in your project:
```
<dependency>
	<groupId>ch.thn</groupId>
	<artifactId>stringutil</artifactId>
	<version>0.0.1-SNAPSHOT</version>
</dependency>
```
